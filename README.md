# Arthur Maine Builder Website

This is a simple mobile-friendly contractor website.

## Files
- `index.html` — main website page
- `styles.css` — design and mobile layout
- `script.js` — mobile menu and year update

## How to use
Upload these files to Vercel, Netlify, GitHub Pages, or any basic web host.

## What to change
- Business phone number
- Real project photos
- Service area
- License/insurance wording
- Any services you do or do not want listed

## Client Portal Link
The website now links to `/portal`. When your software is live, replace `/portal` in `index.html` with the software URL, like `https://app.yourdomain.com`.
